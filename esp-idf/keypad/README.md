## Keypad 4x4

COL1 in the right.
The first and last pins are not used.

| Keypad | wirecolor | GPIO Pin |
| ------ | --------- | -------- |
| COL1   | orange    | 4        |
| COL2   | blue      | 16       |
| COL3   | purple    | 17       |
| COL4   | yellow    | 18       |
| ROW1   | brown     | 19       |
| ROW2   | gray      | 21       |
| ROW3   | green     | 22       |
| ROW4   | black     | 23       |
